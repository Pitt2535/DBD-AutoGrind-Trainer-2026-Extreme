## ⚠️ Troubleshooting common issues

**The menu does not appear in-game.**
Ensure you run the game in Borderless Windowed mode rather than Fullscreen mode. Overlays often fail to render in Exclusive Fullscreen.

**The game crashes upon startup.**
Close any other third-party software that injects code into your games, such as streaming overlays or recording tools, as these can conflict with the trainer.

**The aimbot feels jerky.**
Lower your in-game sensitivity settings. The tool works best when your base mouse speed is set to a moderate level.

**Updates required.**
If an update for Dead by Daylight releases, the trainer might stop working temporarily. Check the releases page for a new version compatible with the current game build.
